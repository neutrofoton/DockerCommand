# Command

1. Run docker compose
    ```
     docker-compose up

     # to start docker compose in background
     docker-compose up -d
    ```

    

2. Open URL based on port in docker-compose.yml
   ```
   # API
   http://localhost:5001

   # Web
   http://localhost:5000

   ```
